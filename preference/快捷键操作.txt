Google Chrome浏览器
     Ctrl - Tab     C-shift-tab    新开标签
     Ctrl - W       关闭
     Ctrl - N        新浏览器窗口
     C-[1-9]  C-9
     以上4个快捷键和切换标签页有关 前两个前后切换  
                                                     第三个切换到指定顺序标签页  第四个切换到最后一页
     安装了KeyBoardNavigation插件  按键“，”显示数字
     只需要输入数字即等同鼠标点击链接
    
     C-f  搜索   C-g 搜索下一条匹配  C-shift-g 上一条匹配
     C-l   高亮显示url
     输入bing tab 关键字  使用其他搜索引擎
     输入网址中心内容  C-enter 补全www和.com

Gnome和nautilos快捷键
ALT + Left/Right：在Nautilus导航文件夹时执行回退/前进操作
ALT + Up/Down：在Nautilus导航文件夹时执行向上/向下操作
ALT + Home：跳转至家目录
C-A-T  ： 启动terminal
ALT+ F1：启动程序菜单，这可能我平时用得最多的
ALT + Enter：显示文件/文件夹属性
ALT + F2：启动“运行应用程序”对话框
ALT + F4：关闭窗口
ALT + Tab : 浏览窗口
Shift + Del：不通过回收站，直接永久删除文件
F2：重命名文件/文件夹
F3：启动Nautilus的分割浏览
F8：显示/隐藏Nautilus菜单栏
F9：切换Nautilus侧栏
Super + Tab：“Shift切换”，需要在CCSM中启动
Super + M：反转所有颜色
Super + A或W：分组排列窗口（需要Compiz）
CTRL + L：浏览/编辑完整位置路径
CTRL + A：全选
CTRL + N：新窗口
CTRL + Shift + N：新文件夹
CTRL + H：显示隐藏文件/不显示隐藏文件
CTRL + ALT + L：锁定屏幕
当然啦，您可以自行指定快捷键。比如我使用CTRL + ALT + M来启动终端。您可以在这里指定快捷键：System – Preferences – Keyboard Shortcuts。
